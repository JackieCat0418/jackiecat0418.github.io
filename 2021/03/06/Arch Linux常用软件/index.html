Arch Linux常用软件推荐

很多人说Linux软件生态不好，其实不然。经过这么长时间的发展，Linux上也有许多优秀的自由软件。其中有着Linux下的windows兼容层——wine,经过某深度公司的优化后的deepin-wine5，强大的开发工具vim等等。但是嘛，很多人不知道有哪些好用的软件，所以才会有这篇文章。

添加ArchLinuxCN仓库

ArchLinuxCN仓库上有许多编译好的二进制软件包，其中有很多实用的工具，肥肠肥肠建议添加

编辑/etc/pacman.conf

这个文件是pacman的配置文件，我们需要编辑它去修改pacman的行为，我们在文件中添加以下内容

[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch
#上面使用的是清华的ArchLinuxCN镜像源，我们可以使用阿里源
#ali： https://mirrors.aliyun.com/archlinuxcn/$arch
#在这里写一个小建议，我们可以把配置文件中的#color的#去掉，这样pacman的输出会有颜色，好看点awaaa

添加后，我们安装archlinuxcn-keyring去添加gpg key以正常下载软件包

最后，执行sudo pacman -Syy刷新仓库缓存，就大功告成了！！

常用软件

截图工具

截图工具这里推荐flameshot，搭配上kde的快捷键设置会有比较好的体验

执行sudo pacman -S flameshot

在终端输入flameshot gui时flameshot的界面就会出现

我们可以通过快捷键设置获得更好的体验

打开kde系统设置，快捷键>自定义快捷键中，点击编辑，添加一个命令型的全局快捷键，触发器那里设置你想用的快捷键，动作当然就是flameshot gui啦

之后，应用，尝试使用快捷键触发flameshot吧！

浏览器

在这里我推荐google-chrome和firefox

如果不想麻烦就使用google-chrome，因为firefox在Linux平台下需要搭配硬件解码才能获得更好的体验，而配置硬件解码比较麻烦，我会再写一篇文章去讲疯狂挖坑

Google-chrome

执行sudo pacman -S google-chrome

等待安装即可

FireFox

执行sudo pacman -S firefox

硬解看我什么时候心情好又有时间就写（懒

或者去看官方wiki上的教程

AUR helper

强推pikaur，这个AUR helper允许你去编辑pkgbuild，当这个文本文件里面的仓库出了错这类的，你可以手动更正

执行sudo pacman -S pikaur

用法：pikaur 你想要安装的包

搭配上tsocks食用更佳

办公

在这里推荐wps-office-cn或者libreoffice-fresh-zh-cn

WPS

在这里我们需要使用AUR helper安装

以pikaur为例，执行pikaur wps

它会列出所有带有这个关键词的AUR包，并给予对应的序号，在这里我们要安装wps-office-fonts、wps-office-mine-cn、wps-office-mui-zh-cn、ttf-wps-fonts和wps-office-cn,虽然看着包多但是他们的服务器都在境内所以不用害怕

输入这些包对应的编号，回车

之后询问你是否安装，继续回车

之后问你是否编辑pkgbuild，这个一般情况请输入n，若遇到安装错误就按照日志输出去修改文件

剩下的就交给pikaur去搞定吧，如果提示输入密码就输入吧

结束后，wps就成功安装了

LibreOffice

执行sudo pacman -S libreoffice-fresh-zh-cn，等待安装完成即可

Wine

执行sudo pacman -S wine wine-mono wine-gecko

安装完后，你应该就可以在文件管理器里面打开*.exe文件了，但是可能会经常遇到报错，一般都是wine兼容不好，这个一般是上游问题，只能等修复，亦或者你没有安装依赖，例如dotnet这类的，最好上网找教程

IM（Internet Messager）

就是拿来聊天的，这里推荐几个吧

Tim

是的，你没看错，我们可以在Linux平台下运行tim

首先，我们去编辑/etc/pacman.conf，把[multilib]和它下面的Include = /etc/pacman.d/mirrorlist都取消注释，之后执行sudo pacman -Syy刷新仓库缓存

之后，就要用上我们万能的AUR了！

在这里，aur上的包会使用某深度公司的deepin-wine5，它的兼容性更强

执行pikaur com.qq.tim.spark

之后应该就会找到这个包，安装它

也许网络会不太好，请安装tsocks软件包，并百度tsocks 文件去查找配置文件编写教程，我还是懒

之后把tsocks添加在命令的最前端就可以强行代理了

安装完成就能用了

WeChat

本来还是不太想写微信的，因为这玩意真的反人类，但是还是有人会需要的，故写了这一段

执行pikaur com.qq.wechat.spark

同上，速度慢就走proxy

安装完就能用

Telegram

安装telegram-desktop软件包即可

Discord

安装discord软件包即可

输入法

输入法在这里推荐fcitx5

安装fcitx5、kcm-fcitx5、fcitx5-configtool和fcitx5-chinese-addons

之后添加环境变量

新建.pam_environment在~目录

输入

GTK_IM_MODULE DEFAULT=fcitx
QT_IM_MODULE  DEFAULT=fcitx
XMODIFIERS    DEFAULT=\@im=fcitx
SDL_IM_MODULE DEFAULT=fcitx

之后注销你的电脑，重新进入桌面（！注意：kde桌面的注销有奇妙的bug,重新进入桌面后可能会出现一些奇奇怪怪的现象，如果不是有什么重要的事情建议用重启代替注销）

fcitx5应该就正常运行了

之后执行fcitx5-configtool

添加拼音输入法，建议开启云拼音

之后在桌面添加输入法面板这个部件，添加后fcitx5的主题将会跟随系统主题

字体

Google字体

这边建议使用noto-fonts-cjk这套字体，这套字体非常好看，我的blog使用的就是这套字体，安装后，某些软件可能会出现日文字体异性的现象

这是因为noto-fonts-cjk是有着中日韩三家的字体，如果开启了ja的lang的话，由于j在z前面，所以会优先使用ja，所以，我们要去设置字体优先级，使得zh-cn优先于ja

在~目录新建.fonts.conf文件，输入

<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
 <alias>
  <family>sans-serif</family>
  <prefer>
   <family>Noto Sans CJK SC</family>
   <family>Noto Sans CJK TC</family>
   <family>Noto Sans CJK JP</family>
  </prefer>
 </alias>
 <alias>
  <family>monospace</family>
  <prefer>
   <family>Noto Sans Mono CJK SC</family>
   <family>Noto Sans Mono CJK TC</family>
   <family>Noto Sans Mono CJK JP</family>
  </prefer>
 </alias>
 <dir>~/.fonts</dir>
</fontconfig>

之后，执行fc-cache -fv刷新字体缓存，，重启你打开的软件,你就会发现这个问题被修复了

媒体播放
