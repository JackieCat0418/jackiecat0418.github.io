ArchLinux KDE桌面环境安装

此文由RS_Apple编写！！！

新建普通用户

在日常生活中我们不提倡使用root账号，因为这会对系统拥有完全的掌控权，容易自己把系统搞坏，所以我们要新建账户

useradd -m -G wheel username

之后，我们要使得我们的账户能够正常使用sudo指令提权，我们要编辑sudoers file

EDITOR=vim visudo
#可以替换为你熟悉的编辑器

找到%wheel ALL=(ALL) ALL
去掉前面的注释，这步就完成了

安装显卡驱动

sudo pacman -S mesa #安装基础显卡驱动，有些软件以它为依赖
sudo pacman -S nvidia #安装N卡驱动(注意，n卡驱动建议搭配bumblebee等软件进行食用，fu*k u nvidia)
sudo pacman -S xf86-video-intel #intel
sudo pacman -S xf86-video-ati #amd

安装KDE-Plasma的必要软件

sudo pacman -S xf86-input-synaptics #安装触摸板驱动(可选，一般不需要安装)
sudo pacman -S noto-fonts-cjk #安装必要的字体,我比较喜欢google家的字体

安装KDE-Plasma桌面环境

sudo pacman -S plasma kde-applications #安装桌面环境以及桌面环境的应用程序
sudo pacman -S sddm sddm-kcm #安装sddm
sudo systemctl enable sddm #开机自动启动sddm登陆管理器

添加archlinuxcn软件源（建议）

archlinuxcn是中国的archlinux用户自己维护的软件源，上面有许多已经打包好的官方源没有的软件包

如果不添加这个源可能需要经常使用aur helper安装软件

我们后面的教程也会用到上面的软件包，使用cn源可以加快你的速度

编辑/etc/pacman.conf文件，加入以下内容

[archlinuxcn]
Server = https://mirrors.aliyun.com/archlinuxcn/$arch
#这里使用了阿里云的镜像源，你也可以选择其他的镜像源
#清华源:https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

重启

这些软件包安装完成后，重启你的电脑，不出意外的话sddm就会先启动

找到之前新建的账户，输入密码，登陆进去，如果看到了kde就证明你成功了！
